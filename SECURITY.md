# Security Policy

## Supported Versions

Security support is provided for the current KingPepe Core v31.1 release series
published in this repository.

## Reporting A Vulnerability

Report security vulnerabilities through this repository's GitHub Security
Advisories when available:

https://github.com/kingpepe2/king-pepe-source-code/security/advisories

If private vulnerability reporting is unavailable, open a GitHub issue without
including exploit details and request a private disclosure channel.

Do not publish private keys, wallet files, RPC passwords, debug logs containing
credentials, or exploitable vulnerability details in public issues.
